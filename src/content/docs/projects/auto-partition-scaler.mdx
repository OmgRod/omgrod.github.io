**Auto Partition Scaler** is a Windows desktop application designed to automatically manage, monitor, and resize disk partitions on your system. It provides a user-friendly interface for configuring auto-resize rules, tracking partition history, and visualizing disk usage over time.

## Key features include:

- **Automatic Partition Resizing:** Set up rules to automatically expand partitions when free space drops below a threshold.
- **Partition History Tracking:** Enable history tracking for partitions to monitor size changes and usage trends.
- **Graphical History Viewer:** Visualize partition usage history with built-in graphing tools.
- **Configurable UI:** Easily add, edit, or remove partition configurations using a sidebar and dialog-based interface.
- **Safety Features:** Prevents accidental modification of the system drive and provides warnings for risky operations.
- **Background Service:** Includes a background process to monitor and manage partitions even when the main UI is closed.
- **Logging and Diagnostics:** Integrated logging for troubleshooting and operational transparency.
